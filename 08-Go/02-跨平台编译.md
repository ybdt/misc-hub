### 选项解释
```
go tool dist list  # 查看支持哪些系统和架构
-ldflags "-s -w"  # 用于缩小体积
```

### 01-Windows下编译Windows、Linux、MacOS
编译windows下amd64和386可执行文件
```
# cmd
set GOOS=windows && set GOARCH=amd64 && go build -ldflags "-s -w" -o windows_amd64.exe main.go
set GOOS=windows && set GOARCH=386 && go build -ldflags "-s -w" -o windows_386.exe main.go

# powershell
$Env:GOOS = "windows"; $Env:GOARCH = "amd64"; go build -ldflags "-s -w" -o windows_amd64.exe main.go
$Env:GOOS = "windows"; $Env:GOARCH = "386"; go build -ldflags "-s -w" -o windows_386.exe main.go
```

编译linux下amd64和386可执行文件
```
# cmd
set GOOS=linux && set GOARCH=amd64 && go build -ldflags "-s -w" -o linux_amd64 main.go
set GOOS=linux && set GOARCH=386 && go build -ldflags "-s -w" -o linux_386 main.go

# powershell

$Env:GOOS = "linux"; $Env:GOARCH = "386"; go build -ldflags "-s -w" -o linux_386 main.go
```

编译macos下amd64和arm64可执行文件
